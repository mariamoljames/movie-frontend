## Movie Frontend

This is the frontend repo of Movie, an app to search for movies and view their details. 

### Tech Stack

* React.js
* Ruby on Rails API
* PostgreSQL DB

### Clone the repo, cd into the folder and run the following commands to start the app.

* npm install
* npm start
